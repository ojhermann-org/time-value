# time_value-cli

The command-line interface for [`time_value`](../time_value) — type-safe
time-value-of-money calculations from your shell. Installs the `time-value`
binary.

> **Status:** scaffolding. The subcommands (`pv`, `fv`, `npv`, `irr`, annuity, …)
> are designed in [ADR-0010](../../docs/adr/0010-cli-surface.md) and land in a
> later phase.
