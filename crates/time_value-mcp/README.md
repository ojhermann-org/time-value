# time_value-mcp

A [Model Context Protocol](https://modelcontextprotocol.io) server exposing
[`time_value`](../time_value)'s time-value-of-money calculations as tools over
stdio. Installs the `time-value-mcp` binary.

> **Status:** scaffolding. The tool surface is designed in
> [ADR-0011](../../docs/adr/0011-mcp-server.md) and lands in a later phase. Async
> is contained to this crate; the core library stays synchronous.
